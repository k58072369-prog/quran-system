🛠️ تطوير النظام بالكامل + إصلاح قسم الحصص + إضافة قاعدة بيانات محلية قوية

المطلوب تطوير النظام الحالي بالكامل بشكل احترافي مع الحفاظ الكامل على جميع الأقسام الموجودة وعدم التأثير على أي وظيفة حالية داخل الموقع.

━━━━━━━━━━━━━━

💾 إضافة قاعدة بيانات محلية قوية

المطلوب إنشاء قاعدة بيانات محلية احترافية لتخزين جميع بيانات النظام، بحيث:

• يتم حفظ أي بيانات جديدة تلقائيًا.
• يتم حفظ أي تعديل أو حذف مباشرة.
• تتحمل حجم بيانات كبير جدًا بدون بطء أو تهنيج.
• تكون منظمة وقابلة للتوسع مستقبلًا.
• تدعم العلاقات بين الجداول بشكل صحيح.
• تمنع فقدان البيانات أو تكرارها.
• تدعم النسخ الاحتياطي واسترجاع البيانات لاحقًا.

يفضل استخدام:
• SQLite أو PostgreSQL محليًا
أو أي نظام قوي مناسب لتحمل البيانات الكبيرة واستقرار الأداء.

━━━━━━━━━━━━━━

👨‍🎓 نظام الطلاب

قسم الطلاب موجود بالفعل ويجب تطويره بحيث:

• إضافة طالب جديد.
• تعديل بيانات الطالب.
• حذف الطالب بشكل آمن.
• ربط الطالب بمعلم محدد.
• إمكانية نقل الطالب بين المعلمين.
• حفظ سجل متابعة الطالب بالكامل.

بيانات الطالب تشمل:
• الاسم
• السن
• رقم الهاتف
• ولي الأمر
• المعلم التابع له
• الحلقة
• مستوى الحفظ
• الملاحظات

━━━━━━━━━━━━━━

👨‍🏫 نظام المعلمين

• إضافة وتعديل المعلمين.
• كل معلم يكون له مجموعة طلاب خاصة به.
• إمكانية عرض جميع الطلاب التابعين للمعلم.
• معرفة عدد الحلقات والحصص الخاصة به.

━━━━━━━━━━━━━━

📚 نظام الحلقات

عند إنشاء حلقة جديدة:

• يتم اختيار المعلم أولًا.
• بعدها يظهر تلقائيًا جميع الطلاب التابعين لهذا المعلم.
• يتم اختيار الطلاب وإضافتهم للحلقة مباشرة.
• حفظ بيانات الحلقة وربطها بالمعلم والطلاب.

الحلقة تحتوي على:
• اسم الحلقة
• المعلم
• الطلاب
• المواعيد
• عدد الحصص
• الملاحظات

━━━━━━━━━━━━━━

🕌 نظام الحصص

يوجد مشكلة حاليًا داخل قسم الحصص، حيث عند إنشاء حصة جديدة وإدخال جميع البيانات بشكل صحيح وتحديد الحلقة والمعلم، لا يتم إنشاء الحصة.

المطلوب:
• إصلاح المشكلة بالكامل.
• التأكد من حفظ الحصة داخل قاعدة البيانات بنجاح.
• ظهور رسالة نجاح واضحة بعد إنشاء الحصة.
• منع أي أخطاء أثناء الحفظ أو الربط.

━━━━━━━━━━━━━━

📋 عرض الحصص

بعد إنشاء الحصة يجب أن تظهر داخل القائمة مع:
• اسم الحصة
• الحلقة التابعة
• اسم المعلم
• التاريخ والوقت
• حالة الحصة
• زر "عرض الحصة"

━━━━━━━━━━━━━━

👨‍🎓 داخل صفحة الحصة

عند الضغط على "عرض الحصة" يتم فتح صفحة متابعة متكاملة للطلاب.

بعد اختيار الحلقة:
• يتم جلب جميع الطلاب التابعين للحلقة تلقائيًا.
• عرضهم داخل جدول منظم وسريع.
